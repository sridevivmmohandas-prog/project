# Auto-Creative Engine

## Overview
Businesses spend weeks designing multiple variations of the same image content for marketing campaigns. The **Auto-Creative Engine** leverages **Generative AI** to automatically generate high-quality ad creative variations, saving time and effort.

**Project Goal:**  
Given a brand's logo and product image, automatically generate **10+ ad creative variations** with high-resolution images and matching captions.

---

## Features
- **Automated Image Variations:** Generates multiple ad creatives from input images.  
- **High-Resolution Output:** Provides ready-to-use images for marketing campaigns.  
- **Caption Generation:** Generates creative ad copy using a language model.  
- **Output Packaging:** Returns a zip file containing all images and captions.  

---

## Technical Requirements
- **Image Generation:** Uses APIs like Stable Diffusion, MidJourney, or DALL-E 3 to generate variations.  
- **Text Generation:** Generates ad captions using a large language model (LLM).  
- **Automation:** Complete pipeline from input images to downloadable zip file.
